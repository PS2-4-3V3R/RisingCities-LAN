# RisingCities-LAN
**This is a combination of Michielvde1253's and glennhenry's solutions**

Michielvde1253 repo: https://github.com/Michielvde1253/risingcities-client.

glennhenry repo: https://github.com/glennhenry/Rising-Cities-Server/tree/main.

 

But written in C#. It uses .NET 8 and WPF for server visualization. (It doesn't get past the loading screen).

 

**This repository includes the executable ready to run on Windows.**

Simply go to http://localhost:8000/play.html in a Flash Player-compatible browser (such as Basilisk) when your server is running.

Basilisk browser: https://www.basilisk-browser.org/

Flash Player (Latest version without time bomb): https://ia903406.us.archive.org/view_archive.php?archive=/1/items/Adobe_Flash_Player_Complete_Collection/fpdownload.macromedia.com/pub/flashplayer/installers/archive/fp_32.0.0.371_archive.zip

Direct download for Flash Player NPAPI: https://archive.org/download/Adobe_Flash_Player_Complete_Collection/fpdownload.macromedia.com/pub/flashplayer/installers/archive/fp_32.0.0.371_archive.zip/32_0_r0_371%2Fflashplayer32_0r0_371_win.exe

Direct download for Flash Player Debug NPAPI: https://archive.org/download/Adobe_Flash_Player_Complete_Collection/fpdownload.macromedia.com/pub/flashplayer/installers/archive/fp_32.0.0.371_archive.zip/32_0_r0_371_debug%2Fflashplayer32_0r0_371_win_debug.exe

NPAPI = win.exe = Firefox (also Basilisk browser)

ActiveX = winax.exe = Internet Explorer (Run in Windows 7 compatibility mode if the installation says you have a newer version)

PPAPI = winpep.exe = Chrome
